# Smart Transactions API

The Smart Transactions API allows developers to read and write data relevant to constructing a smart transaction.

- Intent Batch
    - read
    - write

The API is available at https://smart-transaction.districtfinance.com

## `/intent-batch/read/{QueryParams}`

### QueryParams
- type
- isActive
- isCancelled
- isExecuted

## `/intent-batch/create/{QueryParams}`
